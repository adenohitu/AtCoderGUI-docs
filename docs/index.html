HisuiHisui-競プロ支援ツール-

競プロを簡単に

競技プログラミング専用ツールHisuiを使って、競プロもっと楽しもう
競プロ向けにカスタマイズされたエディター、 順位や時間などの必要な情報をすぐに手に入れられるダッシュボード 競プロに役立つさまざまな機能が備わっています

ドキュメント-準備中- ダウンロード-準備中-

最新版:近日α版公開予定

このアプリケーションはまだ開発段階です

機能・デザインは変更される場合があります

Coding

競技プログラミングに最適化されたエディターが搭載されています
VSCodeのエディターコアとして使われているMonacoEditorを採用
初めての方でも直感的に使うことができます
また、書いたコードをすぐにテストして問題がなければ提出することもできます

Ranking

競技プログラミングではランキングが重要になってきます
ランキング情報をより直感的に見ることのできるダッシュボードが搭載されています

History

これまでにコンテスト中に書いたコード・過去問を解いた時に書いたコードの全てを管理することができます
テストを実行したときや提出したタイミングでGitに自動でCommitされます
またコードをGithubといったサービスにあげてすぐに共有することができます

Test & Submit

書いたコードをすぐにテストして問題がなければアプリ内で提出することができます

Generate testcase

制約を入力することで新たなテストケース生成を生成できます
公開されていないテストケース生成も試すことでバグなどに気付くことができます

Add-on

ユーザーが自分で機能を増やせるインターフェース
特別な知識なしですぐに作り始められます
作った機能はマーケットプレイスを通して公開することで色んな人に使ってもらえます

初心者の大きな壁、環境構築
インストールするだけ

難しい設定などはなく、すぐに使い始めることができます

ライバルに勝ちたい？

必要な情報を選び表示できるダッシュボード
時間のないコンテスト中に素早く順位表などから得られる情報を手に入れることができます

バグを見つける

新たなサンプルケースを生成できる
公開されているサンプルケースをすべて試してもどこにバグがあるかわからないとき、
すぐに新たなサンプルケースを作って試すことができます

自動のコード管理

コードテストテストを実行したタイミングや提出したタイミングでソースコードを保存
コンテストに参加中に間違って消してしまってもすぐに戻すことができます
